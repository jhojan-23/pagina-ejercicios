Pregunta de Selección Múltiple

Pregunta

Pregunta 5: ¿Cuáles de las siguientes afirmaciones son verdaderas sobre la media y la mediana de un conjunto de datos?

Respuestas

Si el conjunto de datos está sesgado a la derecha, la media será mayor que la mediana.

Si el conjunto de datos está sesgado a la izquierda, la media será menor que la mediana.

Si el conjunto de datos tiene una distribución simétrica, la media será igual a la mediana

Si el conjunto de datos tiene valores atípicos, la mediana será más afectada que la media.
